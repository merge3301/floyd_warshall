updateVisualizationNodes

fun updateVisualizationNodes(graphLayer: Pane, matrixInput: MatrixInput, size: Int, highlights: List<Triple<Int, Int, String>>, nodePositions: MutableList<Pair<Double, Double>>, edgeMap: MutableMap<Pair<Int, Int>, Pair<Line, Polygon>>, nodeMap: MutableMap<Int, Circle>, onVertexClicked: (Int) -> Unit, selectedNodes: Collection<Int> = emptyList())

Перерисовывает граф: вершины, рёбра, лупы и лейблы.

Parameters

graphLayer

Слой (Pane), на который всё рисуется.

matrixInput

Матрица смежности (в т.ч. для получения содержимого и текстовых полей).

size

Размер графа (количество вершин).

highlights

Подсветки рёбер/вершин (i, j, тип), например, "added", "candidate".

nodePositions

Список координат вершин (обновляется при drag).

edgeMap

Мапа рёбер (i, j) → (Line, Polygon).

nodeMap

Мапа вершин (i) → Circle.

onVertexClicked

Callback для клика по вершине (передаёт номер вершины).

selectedNodes

Список выделенных вершин (по умолчанию пуст).
