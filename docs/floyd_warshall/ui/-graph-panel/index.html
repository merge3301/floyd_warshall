class GraphPanel(val matrixInput: MatrixInput)

Панель для визуального отображения графа и взаимодействия с ним.

— Рисует вершины и рёбра по матрице смежности. — Реализует drag'n'drop перемещение графа и вершин. — Позволяет добавлять вершины правым кликом. — Синхронизируется с MatrixInput и обновляет граф при изменении матрицы.
