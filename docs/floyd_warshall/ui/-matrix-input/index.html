MatrixInput

class MatrixInput

Виджет для ввода и отображения матрицы смежности графа.

Позволяет вручную или случайно задавать матрицу, выбирать размер графа и количество рёбер, а также подсвечивать ячейки и заголовки для визуализации работы алгоритма.
